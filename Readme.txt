Para execução do projeto siga as instruções abaixo:

1 - Instalação do ruby na versão 3.0.2-1, você pode encrontrá-lo nesse site: https://rubyinstaller.org/downloads/ (instalação next next finish)

2 - Executar linha de comando gem install bundler

3 - No cmd acessar a pasta do projeto executar o comando bundle install

4 - Para exutar os casos de teste executar o comando rspec --format html --out log/report.html -fd

Na pasta log serão criados os relatórios de execução
